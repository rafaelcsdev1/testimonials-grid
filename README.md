
# Testimonials Grid Section

![Preview do projeto](./preview.jpg)

## Descrição

Este projeto é uma solução para o desafio [Testimonials Grid Section](https://www.frontendmentor.io/challenges/testimonials-grid-section-Nnw6J7Un7) do Frontend Mentor. O objetivo é construir uma seção de depoimentos em grid, responsiva e fiel ao design proposto.

## Tecnologias Utilizadas

- HTML5
- CSS3 (incluindo Grid e Flexbox)

## Funcionalidades

- Layout responsivo: se adapta para desktop e mobile (375px)
- Grid de depoimentos com diferentes estilos para cada card
- Imagens otimizadas e assets fornecidos pelo desafio

## Como rodar o projeto

1. Clone o repositório:
	```
	git clone https://github.com/rafaelcsdev1/testimonials-grid.git
	```
2. Abra o arquivo `index.html` no seu navegador.

## Aprendizados

- Prática com CSS Grid para layouts complexos
- Uso de Flexbox para centralização e alinhamento
- Adaptação de layouts para diferentes tamanhos de tela usando media queries
- Atenção a detalhes de espaçamento, tipografia e cores

## Desafios encontrados

- Ajustar o posicionamento do SVG decorativo no card principal
- Garantir que o grid ficasse visualmente agradável em telas pequenas
